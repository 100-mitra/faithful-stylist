The Faithful Stylist

A grounded jewellery recommender. Every factual claim (price, metal, stone, carat) is templated from the catalog and verified — within the templated path the model has no value channel for facts, and an in-vocabulary fabrication is blocked before it reaches you. Opinion is always labelled as opinion.

This block is a structural guarantee on the templated path, demonstrated against a self-authored attack set — not a proof against every phrasing (e.g. out-of-vocabulary terms). See README → Limitations.
